# W3FS Chain Explorer

Notice：Node v14.16.1

## Project setup
```
npm install
```

### Modifying the Configuration File (public/link.json)
```
coincap：公共api用来获取其他币的信息
CHAIN_RPC：区块浏览器所在链的地址
Contact：配置联系方式
```

npm run serve
